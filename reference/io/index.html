    styles ¤

    ANSI colors for ThLun library.

    This module provides ANSI escape codes for foreground and background colors, as well as text styles. It supports 256-color terminals.

    Classes:

    Name Description
    Back

    Background colors using ANSI 256-color codes.
    Colors

    Named ANSI color codes (0-255).
    Fore

    Foreground colors using ANSI 256-color codes.
    Style

    ANSI text styles.

    Functions:

    Name Description
    bg_replacer

    Return ANSI escape sequence for 256-color background.
    fg_replacer

    Return ANSI escape sequence for 256-color foreground.

    Back ¤

    Background colors using ANSI 256-color codes.

    Usage

    print(Back.BLUE + "Hello" + RESET)

    Named ANSI color codes (0-255).

    Fore ¤

    Input/Output operations for ThLun library.

    This module provides cross-platform utilities for single-character input, masked (secret) input, and type-restricted character scanning.
